Currently, the four boxes are static, and I've applied background color, border, padding, margin, as well as height and width. Since they are static, CSS properties like top and right won't have any effect.

To move my div elements, I will add the position: relative property, allowing me to use properties like top, right, and left. If I set top: 3px, the box will move down, and if I set right: 3px, the box will move to the left.

If you want your lower div to appear above the upper div, you should apply z-index: 1 to that div. For the z-index to work, the div must have the position: relative property. Yes, this is correct. The z-index property works when an element has a position set to anything other than the default static (like relative, absolute, or fixed).
